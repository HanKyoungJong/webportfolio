포트폴리오 방향

이 포트폴리오는 다음 기술 스택을 기반으로 정리했습니다.

Backend : Java, Spring Boot, Spring MVC, JAVA17

Template Engine : Thymeleaf

Architecture : DDD 스타일 구조

Infrastructure : AWS EC2

Container : Docker

CI/CD : Jenkins

Version Control : GitHub

웹 포트폴리오의 디자인과 사용자 경험을 유지하면서,

Spring Boot + Thymeleaf 기반 서버 사이드 애플리케이션 구조로 재구성한 포트폴리오입니다.

실제 서비스 환경을 고려하여 AWS EC2, Docker, Jenkins 기반 CI/CD 환경을 구축하였으며,

DDD 스타일 구조를 적용하여 경력기술서와 프로젝트 경험을 체계적으로 정리한 백엔드 중심 포트폴리오입니다.
